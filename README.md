### 编程流程

```mermaid
graph TB
a(设计图分析)
b(项目目录组织)
c(编写HTML代码)
d(编写CSS样式)
d-a(PC端样式)
d-b(移动端样式)
e(添加响应式图片)
f(兼容性测试及调试)
g(自动化打包)
a --> b --> c --> d --> d-a --> e --> f --> g
d --> d-b --> e
```
